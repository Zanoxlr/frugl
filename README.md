# SubSmart (working title) — B2C subscription advisor POC

Mobile-first app: see all your utility subscriptions in one place, talk to a grounded AI
advisor to understand what you actually have, define your real needs, and get a right-fit
offer without being upsold. Monetized per honest lead.

- Verticals: telco (real data), energy=electricity+gas (real), insurance (real), water (info-only).
- LLM: `claude -p` (OAuth subscription). Swappable invocation → voice phase 2.
- PUBLIC-SAFE: no prod secrets, no commission data, no CalculatorPlatform engine. Seeded dataset only.
